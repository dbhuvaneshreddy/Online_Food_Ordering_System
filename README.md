# # 🍔 Online Food Ordering System

A simple **Online Food Ordering System** built using **Python** and modular programming. This project allows users to log in, view the food menu, place orders, generate bills, apply coupons, and make payments.

## 🚀 Features

* 🔐 User Login
* 🍕 Food Menu Display
* 🛒 Food Ordering
* 🧾 Bill Generation
* 💰 GST Calculation
* 💳 Cash, UPI, and Card Payment Options
* 🎟️ Coupon Code Support
* 📜 Order History

